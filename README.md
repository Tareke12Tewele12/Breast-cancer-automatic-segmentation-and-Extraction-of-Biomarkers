This project focuses on segmentation models applied before and after treatment.
Subsequently, different biomarkers were extracted from the regions of interest. 
These biomarkers include Standardized Uptake Value (SUV), Metabolic Tumor Volume (MTV), and Total Lesion Glycolysis (TLG).
In addition, we developed a quality control system to carefully identify and analyze potential outliers
Different segmentation models, including 2D, 3D, and nnU-Net architectures, were implemented to perform image segmentation in both 2D and 3D settings
